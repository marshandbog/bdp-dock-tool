BDP Dock Pump Test Tool

Serial diagnostics for the RV2900WD dock — pumps, valves, heaters, sensors, and MIDI, per BDP Command Spec V2.

⬇ Download for Windows

Windows 10/11 · 64-bit · free · installs in seconds

What it does

  • Connect to the dock over any COM port (115200-8N1) and drive individual tests.
  • Query sensors and actuate pumps, valves, heaters, and power rails.
  • Send a MIDI file over the port — raw dump or play-as-MIDI.
Heads up: the installer isn't code-signed yet, so Windows SmartScreen may say "unknown publisher." Click More info → Run anyway. Prefer to build it yourself? Grab the source and run pyinstaller packaging/bdp_tool.spec.
